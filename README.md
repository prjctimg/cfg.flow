# cfg.flow

This repository houses the configuration for my Flow Control setup.

This is WIP but I am trying to make this editor my daily driver since Neovim (though powerful) is starting to feel bloated.

Flow changes this by having a zero config approach where I would normally use a Lua plugin.
Though the editor is still lacking some 'nice to have' features, it is the best way to quiet down the information overloaded programming environments we have today.

Can't wait to see what v1 will bring...
